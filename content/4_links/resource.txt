Title: Links

----

Text:

This is a loose collection of links, some useful, some fun, some inspirational. It is mainly a resource for my students and is updated infrequently. I hope you find it useful!

**Coding**
(Link: https://github.com/EbookFoundation/free-programming-books Text: Free programming books)
A huge collection of free programming books on github

(Link: https://www.freecodecamp.org/ Text: Free Code Camp)
Learn to code for free in a camp

(link: https://www.codecademy.com/ text: Codecademy)
Learn to code interactively, also for free

(Link: https://developer.mozilla.org/en-US/docs/Learn Text: Mozilla Developer Network)
A good start for complete beginners

(link: https://www.kevinpowell.co text: Kevin Powell | CSS is Awesome)
Don't let the bio fool you, Kevin is a great teacher.

(link: https://getkirby.com text: Kirby CMS)
A file based CMS that is easy to use and learn.

(Link: https://readings.design/PDF/Design-Interface.pdf Text: Design Interface)
A book on design and code by Mikaella Clements and Matej Latin

**Foundries**
(Link: https://typefoundry.directory/ Text: Directory of Typefoundries)
This is a very useful collection of many many typefoundries with some extra information about licensing. 

(Link: https://type-foundries-archive.com/ Text: Typefoundries Archive)
Another and perhaps the most complete collection of foundries worldwide

(Link: https://www.typotheque.com/ Text: Typotheque)
Based in the Netherlands, extreme range and always a good source for reading

(Link: https://velvetyne.fr/ Text: Velvetyne)
French foundry with a very nice selection of open source fonts

(link: https://boldmonday.com/ text: Bold Monday)
Solid Dutch type design

(link: https://hardtype.xyz text: Hard Type)
My favourite little boutique for new, fresh Swiss type design

(link: https://abcdinamo.com/ text: Dinamo)
Swiss type design and funk with the best webfonts test site

(Link: https://ohnotype.co Text: Oh No Type)
If you're looking for fun display fonts, this is the place to be, based in the USA


**Font Platforms**
(link: https://www.futurefonts.xyz/ text: Future Fonts)
Collection of foundries with fonts in the making

(link: https://fontstand.com/ text: Fontstand)
Find, try, rent or buy fonts from independent foundries 


**Inspiration**
(Link: https://oa.letterformarchive.org/ Text: Letterform Archive)
THE Letterform Archive, a real treasure

(Link: https://archives.design/ Text: Archive Design)
Collection of graphic design items available online

(link: https://www.hoverstat.es/ text: Hoverstates)
Archive of great web design examples


**Technology**
(Link: https://caniuse.com/ Text: Can I use)
What works where and why, current browser support tables

(Link: https://pixelambacht.nl/chromacheck/ Text: Chroma Check)
Perfectly simple check for support of colour font tables in browsers

(Link: https://addons.mozilla.org/en-US/firefox/addon/whatruns/ Text: What runs)
Small addon for Firefox to see what run's on a website

(Link: https://namecheck.fontdata.com/ text: Check font names)
If you are designing a font and wonder if the name is already taken, this is a good place to check.


**Identity**
(link: https://brandingstyleguides.com/ text: Branding Styleguides)
A large and broad collection


**Publishers**.
(Link: http://booooooks.scrgb.co/ Text: Overview of independent publishers)
Not much else to add

(link: https://www.inventorypress.com text: Inventory Press)
Inventory Press publishes books with an emphasis on subcultures and socio-political aspects of culture.

(Link: https://www.lespressesdureel.com/EN/nouveautes.php Text: Les presses du r√©el)
Large French publishing house

(Link: https://editiontaube.de/de Text: Edition Taube)
A small publisher from Zurich


**Magazines**
(Link: https://www.macguffinmagazine.com/ Text: MacGuffin magazine)
Researched Design & Craft

(Link: https://worksthatwork.com/ Text: Works That Work)
Essays on the impact of creative work in everyday life

(link: http://www.eyemagazine.com/ text: Eye Magazine)
Quarterly graphic design magazine from London

(link: https://lapolice.ch/publications/ text: Footnotes Magazine)
Periodical type design magazine from La Police


**Read**
(Link: https://readings.design/ text: Readings Design)
A collection of essays and articles on design

(Link: https://letterror.com/articles/bad-type.html Text: Bad Type)
Article about defining quality criteria for all type by Erik van Blokland

(Link: https://fontstand.com/news/knowledge/evaluating-the-quality-of-a-typeface/ Text: Type Quality)
Good article with some handy criteria for evaluating typefaces by Indra Kupferschmid

(Link: http://kupferschrift.de/cms/2012/03/on-classifications/ Text: Critical view on typeface classification)
The boring topic of font classification with a historical context from a critical point of view by Indra Kupferschmid

(Link: https://klim.co.nz/blog/signifier-design-information/ Text: Design Process of Signifier)
Indepth documentation of the design process by Kris Sowersby, see also his other articles

**Listen**
(Link: http://www.typeradio.org/ Text: Type Radio)
"Type is speech on paper. Typeradio is speech on type". 

(link: https://99percentinvisible.org/ text: 99% Invisible)
About things we don't think about, design in the broadest sense


**Watch**
(link: https://vimeo.com/channels/1364477 text: Herb Lubalin lecture series)
Part of the Type@Cooper programme

(link: https://www.youtube.com/channel/UC3TabHJUsa2BkEkN_xSpMvQ text: ATypeI)
Archived talks from the AtypI conference

(Link: https://vimeo.com/showcase/letterform-lectures text: Letterform Lectures)
Complementary series to the Type West type design programme

(Link: https://vimeo.com/istype text: IS Type)
Archived talks from the IS Type conference

(link: https://mnemosyne.esad-amiens.fr/ text: Esad Amiens)
The school's archive of past programmes

(link: http://printingfilms.com/ text: Print Films)
Collection of old films showing the technologies and processes of printing


**Manifesto**
(Link: https://conditionaldesign.org/manifesto/ Text: Conditional Design Manifesto)
A manifesto for artists and designers
